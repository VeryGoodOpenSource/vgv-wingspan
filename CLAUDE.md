# Wingspan

Wingspan, in this repository, is a collection of AI-assited engineering tools, such as skills, agents, subagents, hooks and more with a strong specialization on Flutter and Dart projects released as a Claude Code plugin marketplace.

## Philosophy

Wingspan's reason to exist is to appy VGV's collection of best practices and standards identified while building scalable Flutter apps to our AI-assisted workflows.

The goal is that each step of the development cycle should contribute to make subsquent steps clearer and closer to the final intent of the user, where code and products created excel for their quality. Build the right thing, built the thing right.
